Steps to run:

1) First go to mongo shell.
2) Copy full path of "final_component_db.js" file present under this project.
3) Write load command in mongo shell to execute this script file.
Example: load("<full path of final_component_create_db.js>");
It will print true on successful execution.
4) now run the application with nodemon app command/ node app.js command.
